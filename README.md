# Shumani Raludzingana | Portfolio

A sleek, modern, and responsive professional portfolio showcasing software development projects, technical skills, and experience.

## 🚀 Features

- **Dynamic Typing Effect**: Engaging hero section with auto-typing professional titles.
- **Glassmorphism Design**: Modern UI with frosted glass effects and smooth transitions.
- **GitHub API Integration**: Automatically fetches and displays the latest repositories.
- **Functional Contact Form**: Integrated with Formspree for direct email notifications.
- **Dark/Light Mode**: User-selectable theme with persistent settings.
- **Responsive Layout**: Optimized for all devices from mobile to ultra-wide monitors.
- **Professional Experience Section**: Detailed timeline of career highlights and internships.
- **Copy-to-Clipboard Email**: Quick contact feature in the footer.

## 🛠 Tech Stack

- **HTML5**: Semantic structure and modern accessibility.
- **Vanilla CSS3**: Custom properties, Flexbox/Grid layouts, and CSS animations.
- **JavaScript (ES6+)**: DOM manipulation, Intersection Observer API, and Fetch API.
- **GitHub API**: Dynamic data fetching for projects.
- **Formspree**: Backend-less contact form handling.

## 📁 Project Structure

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Raludzingana98/MyPortfolio.git
   ```
2. **Open `index.html`**:
   Simply open the `index.html` file in any modern web browser to view the site locally.

3. **Configure Contact Form**:
   To make the contact form work for your own email:
   - Create a free account at [Formspree](https://formspree.io/).
   - Replace the `action` ID in the `<form>` tag in `index.html` (line 202).
